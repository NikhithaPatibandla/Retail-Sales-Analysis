# 📊 Retail Sales Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing retail sales data using **SQL**, **Python**, and **Power BI** to uncover meaningful business insights.

The goal is to clean the data, perform structured analysis, and create interactive visualizations that help understand sales performance, profit trends, and regional distribution.

This project demonstrates the ability to work across the complete data analysis pipeline — from raw data to actionable insights.

---

## 🛠️ Tools & Technologies Used

* **SQL (MySQL Workbench)** – Data cleaning and querying
* **Python (Pandas, Matplotlib/Seaborn)** – Data analysis and visualization
* **Power BI** – Interactive dashboard creation
* **Google Colab** – Python execution environment

## 🔎 Project Workflow

### 1️⃣ Data Cleaning (Python)

* Removed duplicates
* Handled missing values
* Converted data types
* Standardized column formats
* Created calculated fields

### 2️⃣ Exploratory Data Analysis (Python)

Visualizations created using Matplotlib and Seaborn include:

* Sales by category
* Profit by region
* Sub-category performance
* Profit margin distribution
* Comparative performance analysis

### 3️⃣ Business Queries (SQL)

* Total sales and profit aggregation
* Region-wise and category-wise performance
* Structured queries for business reporting

### 4️⃣ Dashboard Visualization (Power BI)

* Sales by Category & Sub-category
* Profit distribution by Region
* KPI cards (Total Sales, Total Profit, Profit Margin)
* Interactive filtering and slicers

---

## 📈 Key Insights

* Certain sub-categories generate high sales but low profit margins.
* Regional performance varies significantly in profitability.
* A few product segments contribute disproportionately to overall profit.
